Extract text from image using python program with tesseract

Extract text from image is known as OCR (Optical Character Recognition). Python-tesseract is an optical character recognition (OCR) tool for Python

For Windows download the tesseract from here

You must have python installed in your computer. Then install these modules

pip install opencv-python
pip install pytesseract
pip install tesseract

First import necessary modules

import cv2
import pytesseract

Then read the image and add the location of the tesseract

img = cv2.imread('quotes.jpg')
pytesseract.pytesseract.tesseract_cmd = 'C:/Program Files/Tesseract-OCR/tesseract.exe'

Then convert the image to string using the image_to_string() method . And print the output. The complete code is here.

import cv2
import pytesseract

img = cv2.imread('pic.jpg')
